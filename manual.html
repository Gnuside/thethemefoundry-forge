What is Forge?

Forge is a WordPress theme development toolkit that gives you the power to quickly and easily develop your own WordPress themes in a tidy environment using powerful tools like SASS and CoffeeScript.

How does it work?

Forge installs as a Ruby gem and allows you to quickly bootstrap a brand new theme and work inside a clean development environment with tools like SASS and CoffeeScript. Your theme is then automatically compiled to all your local WordPress environments as you save and work.

Basic setup

  1. Install Ruby and Ruby gems.
  2. Install Forge gem install forge
  3. Create your project forge
  4. Specify your WordPress install(s) forge link /path/to/wordpress
  5. Watch for changes forge watch
  6. Start developing!
