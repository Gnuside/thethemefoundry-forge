What is Forge?

Forge is a WordPress theme development toolkit for bootstrapping and developing WordPress themes in a tidy environment using powerful front-end language tools like SASS and CoffeeScript.

How does it work?

Forge creates a neatly organized project folder with some clean and simple scaffolding (base template files, SCSS files, and theme options). The folder is automatically compiled to your local WordPress install(s) as you save changes and work on your theme. When you are ready to distribute your theme Forge will build it to a folder or package it up into an easy to install zip file.

Basic setup

  1. Install Forge (requires Ruby and Rubygems). 
    gem install forge
  2. Create your new theme project. 
    forge init themename
  3. Tell Forge about your WordPress install(s). 
    forge link /path/to/wordpress
  4. Watch for changes and start developing! 
    forge watch
