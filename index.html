What is Forge?

Forge is a WordPress theme development toolkit for bootstrapping and developing WordPress themes in a tidy environment using powerful front-end language tools like SASS and CoffeeScript.

How does it work?

Forge creates a clean and organized WordPress theme development folder for you to work from. The folder is then automatically compiled to all your local WordPress installs as you save changes to your theme. When you are ready to distribute your theme Forge will build it into a directory or package it up into an easy to install zip file.

Basic setup

  1. Install Ruby and Ruby gems.
  2. Install Forge gem install forge
  3. Create your project forge init
  4. Specify your WordPress install(s) forge link /path/to/wordpress
  5. Watch for changes forge watch
  6. Start developing!
